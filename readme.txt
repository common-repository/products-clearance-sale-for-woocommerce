Products Clearance Sale for WooCommerce is a WooCommerce compatible plugin which enables you to automatically backup existing products and launch them on right time with updated price, description and images. It is certainly an attraction for the earlybird customers.

== Installation ==
To use Products Clearance Sale for WooCommerce, you will need:
* 	an installed and configured copy of [WordPress][] (version 3.0 or later).
*	FTP, SFTP or shell access to your web host

== Screenshots ==

= New Installations =

Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin

Method-B:

1.	Download the Products Clearance Sale for WooCommerce installation package and extract the files on

	your computer. 
2.	Create a new directory named `Products Clearance Sale for WooCommerce` in the `wp-content/plugins`

	directory of your WordPress installation. Use an FTP or SFTP client to

	upload the contents of your Products Clearance Sale for WooCommerce archive to the new directory

	that you just created on your web host.
3.	Log in to the WordPress Dashboard and activate the Products Clearance Sale for WooCommerce plugin.

== Frequently Asked Questions ==

= Upgrades =

To *upgrade* an existing installation of Products Clearance Sale for WooCommerce to the most recent release:

1.	Download the Products Clearance Sale for WooCommerce installation package and extract the files on your computer. 
2.	Upload the new PHP files to `wp-content/plugins/products-clearance-sale-for-woocommerce`,	overwriting any existing Products Clearance Sale for WooCommerce files that are there.
3.	Log in to your WordPress administrative interface immediately in order to see whether there are any further tasks that you need to perform to complete the upgrade.
4.	Enjoy your newer and hotter installation of Products Clearance Sale for WooCommerce
